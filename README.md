# Python-APIs-with-MongoDB
Sample Web app demonstrating creating Python APIs with mongoDb database and Flask


Prerequisites for the Project:

1. [Python](https://www.python.org/downloads/) 
2. [Flask install](https://pypi.python.org/pypi/Flask)
3. MongoDb need to be installed. Follow this link for [guide](https://docs.mongodb.com/manual/administration/install-community/)


It shows how to create diffrent task for a reminder WebApp and save them to mongoDb.Apart from this It provide other
functionalities like Editing, deleting tasks from database;
